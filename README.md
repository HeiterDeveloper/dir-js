# dir-js
File manager written in JS and PHP

```html
<script src="jquery-3.5.1.min.js"></script>
<link type="text/css" rel="stylesheet" href="dir-js.css">
<script src="dir-js.js"></script>

<input type="text" />

<script>
  $("input").dirJs({
  lockTypes: ['image/png', 'application/pdf']
  });
</script>
```
